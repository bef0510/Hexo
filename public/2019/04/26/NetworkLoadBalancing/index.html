NLB 網路負載平衡

發表於 2019-04-26 | 分類於 MIS

網路負載平衡設定

  1. A server IP ： 192.168.1.101
  2. B server IP ： 192.168.1.102
  3. 叢集 IP ： 192.168.1.103

A server 管理 B server

在 A server
[伺服器管理員] →[管理] →[新增伺服器]
[立即尋抓] B server

B server 開啟 網路負載平衡 功能

[伺服器管理員] →[管理] →[新增角色及功能]
選 B server [下一步]
[功能] → 勾選 [網路負載平衡]
[安裝]

A server 開啟 網路負載平衡 功能

重覆 B server 動作，改選 A server

叢集設定

在 A server 開啟 [網路負載平衡管理]
A server 設定

  1. 在 [網路負載平衡叢集] 按右鍵 →[新增叢集]
  2. 主機：輸入 192.168.1.101 →[連線]
  3. [下一步] (不要點其他地方，[下一步] 會不見，Server 2019 也還沒修好)
  4. [優先順序] 選 1
  5. [下一步]
  6. 新增 叢集 IP 192.168.1.103
  7. [下一步]
  8. [叢集操作模式] 選 [多點傳送] [下一步] (虛擬機只能選多點，實體機能選單點、多點)
  9. 等漏斗跑完就完成 [交集]

B server 設定

  1. 在 IP 192.168.1.100 按右鍵 →新增主機到叢集
  2. 主機：B server 192.168.1.102 →連線 (因遠端，會比較久，要有耐心)
  3. 下一步 (不要點其他地方，下一步會不見，把生命浪費在有意義的地方)
  4. 優先順序 選 2
  5. 下一步
  6. 完成
  7. 等漏斗跑完就完成 [交集]

A B server 已 [交集] 後，兩台 server 都要重開機

DNS server 設定

  1. 新增主機
  2. 輸入名稱
  3. 輸入 IP 192.168.1.100
