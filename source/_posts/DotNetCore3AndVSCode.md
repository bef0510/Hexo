## .Net Core 3
1. 開啟 **VS Code**

2. 下載 **C#**
![Architecture](1.png)

3. 關閉 **VS Code**

4. 新增專案
~~~ bash
dotnet new webapi
~~~

5. **VS Code** 開啟專案
~~~ bash
code .
~~~

6. 執行專案
~~~ bash
dotnet run
~~~
